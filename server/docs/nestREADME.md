##Дока по несту

Создание модуля\
```npx nest g mo db```\

Создание сервиса\
```npx nest g s db```\

Создание контроллера\
```npx nest g co db```\

Создание гварда\
используется для ограничения доступа\
```npx nest g gu auth --no-spec --flat```

Создание в текущей директории без тестов\
```npx nest g s password --no-spec --flat```\
